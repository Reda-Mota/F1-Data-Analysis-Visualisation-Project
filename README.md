# 🏎️ F1 Historical Analysis Dashboard

![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white) ![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white) ![Pandas](https://img.shields.io/badge/pandas-%23150458.svg?style=for-the-badge&logo=pandas&logoColor=white) ![Plotly](https://img.shields.io/badge/Plotly-%233F4F75.svg?style=for-the-badge&logo=plotly&logoColor=white)

An interactive web-based dashboard built with **Streamlit** to explore and analyze Formula 1 historical data (1950 - 2025). This project showcases data cleaning, aggregation, and professional visualization techniques.

---

## 🌟 Key Features

- **Professional Sidebar Navigation:** Easy access to different analysis sections (Home, Team Analysis, Circuit Analysis).
- **Real-time Metrics (KPIs):** Instant insights into Ferrari's total wins, the King of Monaco, and track records.
- **Interactive Visualizations:** 
  - **Ferrari Performance:** A time-series line chart showing the evolution of Ferrari's wins over decades.
  - **Monaco Masters:** Horizontal bar chart highlighting the top 5 legends of the Circuit de Monaco.
- **Dynamic Data Explorer:** Fully searchable and filterable raw dataset directly on the Home page.
- **Professional UI/UX:** Clean, structured interface with organized tabs and containers for an optimal viewing experience.

## 🛠️ Tech Stack

- **Frontend Framework:** Streamlit
- **Data Manipulation:** Pandas
- **Data Visualization:** Plotly Express
